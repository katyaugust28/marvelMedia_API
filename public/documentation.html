Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by name /movies/[title] GET None A JSON object holding data about a single movie,
Example:
{
id: 1,
title: 'Iron Man',
year: '2008'
}
Add a movie /movies POST A JSON object holding data about the movie to add,
Example:
{
title: 'Iron Man 2',
year: '2010'
} 		A JSON object holding data about the movie that was added, including an ID:
{
id: 2, title: 'Iron Man 2',
year: '2010'
}
Update the year of a certain movie /movies/:[title]/:[year] PUT None A text message indicating whether the year of the certain movie was updated
Get data about a genre by name/title /movies/genres/:[genre] GET None A text message indicating whether the movie genre is successfully shown
Get data about a director /movies/directors/:[director] GET None A text message indicating whether the movie director is successfully shown
Allow a new user to register /users POST A JSON object holding data about the user,
Example:
{
username: 'katymovies',
password: '1234'
} 		A JSON object holding a message saying the user information was added:

User profile successfully created.
Allow user to update their information /users/:[username] PUT None A text message indicating the profile information was succcessfully updated
Allow user to add a movie to their list /users/:[username]/favorites POST *NEED TO ADD INFO* *NEED TO ADD INFO*
Allow user to remove a movie from their list /users/:[username]/favorites/:[title] DELETE None A text message indicating the movie was removed from their list
Allow user to deregister /users/:[username] DELETE None A text message indicating the user successfully deregistered
